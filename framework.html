The Framework

A structured model for assessing fairness in value chains.

Overview

The Fair Score Framework evaluates fairness across multiple dimensions, enabling organisations to identify imbalances, assess decision‑making processes, and improve equitable outcomes.

Core Components

  • Fairness dimensions
  • Value chain mapping
  • Stakeholder impact assessment
  • Scoring methodology
  • Interpretation guidelines

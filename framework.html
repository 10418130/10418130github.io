The Framework

A structured model for assessing fairness in value chains.

Overview

The Fair Score Framework evaluates value chain fairness across multiple dimensions, enabling the identification of imbalances, assessment of decision‑making processes, and achievement of more equitable outcomes.

Core Components

  • Fairness dimensions
  • Value chain mapping
  • Stakeholder impact assessment
  • Scoring methodology
  • Interpretation guidelines

The process for applying the Framework is outlined on the How It Works page.

How It Works

A simple, structured process for evaluating fairness.

Step‑by‑Step Process

  1. Define the value chain — Identify actors, flows, and decision points.
  2. Assess fairness dimensions — Apply the framework’s criteria.
  3. Score each dimension — Use the structured scoring method.
  4. Interpret results — Identify imbalances and improvement areas.
  5. Implement changes — Use insights to guide fairer decisions.

The methodology is designed to be transparent, replicable, and grounded in empirical research.
